# Should I Watch

Calculate a weighted Rotten Tomatoes, Metacritic, and IMDB collective rating to answer the common "should I watch?" question.

Check out <shouldi.watch> or <shouldiwatch.herokuapp.com>.

## Installation

* `git clone` this repository
* `npm install`
* `bower install`

## Running

* `ember server`

## Running Tests

* `ember test`
* `ember test --server`
